Collection of various Apache Ant tasks

Package Specification

Dependencies to some of thrird party libraries:
  • Sun JRE 1.4.2+
  • Apache Ant 1.6.2+
  • Jakarta Velocity 1.4+
  • Apache Log4J 1.2.8+
  • Commons Logging 1.0.3+
